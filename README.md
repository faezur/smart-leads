# Smart Leads CRM

A full-stack Lead Management System built with React, TypeScript, Express, Prisma, and PostgreSQL.

## Features

- Add new leads
- Update lead status
- Delete leads
- Dashboard statistics
- CSV export
- Form validation
- Responsive clean UI

---

# Tech Stack

## Frontend
- React
- TypeScript
- TailwindCSS
- Axios
- Zustand

## Backend
- Express
- TypeScript
- Prisma ORM
- PostgreSQL
- Zod Validation

## Database
- PostgreSQL (Neon)
